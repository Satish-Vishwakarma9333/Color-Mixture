# 🎨 Color Mixture

A simple and interactive web application that allows users to select two different colors from a custom dropdown interface and generate a visually blended output. The app uses RGB color mixing logic, where the red, green, and blue values of both selected colors are averaged to create a new resulting color. This helps demonstrate how digital color blending works in a simple and visual way, making it both educational and user-friendly.

## ✨ Features

- 🎨 Select two colors from custom dropdown menus
- 🔀 Mix colors using RGB averaging logic
- 🖌️ Live visual preview of mixed color
- 📱 Responsive and modern UI design
- 🧊 Glassmorphism-inspired interface

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glassmorphism UI)
- JavaScript (Vanilla JS)
